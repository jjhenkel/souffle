37 /**
38  * @class MakeIndexTransformer
39  * @brief Make indexable operations to indexed operations.
40  *
41  * The transformer assumes that the RAM has been levelled before.
42  * The conditions that could be used for an index must be located
43  * immediately after the scan or aggregate operation.
44  *
45  * ~~~~~~~~~~~~~~~~~~~~~~~~~~~
46  * QUERY
47  * ...
48  * FOR t1 IN A
49  * IF t1.x = 10 /\ t1.y = 20 /\ C
50  * ...
51  * ~~~~~~~~~~~~~~~~~~~~~~~~~~~
52  *
53  * will be rewritten to
54  *
55  * ~~~~~~~~~~~~~~~~~~~~~~~~~~~
56  * QUERY
57  * ...
58  * SEARCH t1 IN A INDEX t1.x=10 AND t1.y = 20
59  * IF C
60  * ...
61  * ~~~~~~~~~~~~~~~~~~~~~~~~~~~
62  */
